# 概要
最大クリークを求めるための bron kerboschアルゴリズムの並列化。学部の卒業研究として発表できるレベルを目指す。

## 目標
自分で理解した並列化手法を用いてbron kerboschアルゴリズムを並列化し、定量的に性能評価する。
グラフを用いて結果を発表できるようにする。

## できるようにすること

- bron kerboschアルゴリズムの基本である。pivot最適化を使ったtomitaのアルゴリズムを説明できるようになる。
- 単体実装を動かしてみる。
- アルゴリズムを説明できるようになる。
- アルゴリズムの性能を測れるようになる。
- 研究論文に載せられるような大きなデータを用意して実行してみる。
- 並列処理を動かしてみる。
- 並列処理を説明できるようになる。
- 並列処理の性能を評価できるようにする。
- 単体実装との違いをグラフなどで視覚的に説明する。
- なぜ、この結果になるのかの考察が立てられるようにする。
