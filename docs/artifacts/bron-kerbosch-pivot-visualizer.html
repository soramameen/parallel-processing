Bron–Kerbosch 基本形 vs pivot 版

同じグラフで、基本形と pivot 版の探索の違いを比較できます。

「次へ」で開始
BronKerbosch1(R, P, X):
  if P empty and X empty:
    report R
  for v in P:
    BronKerbosch1(R∪{v}, P∩N(v), X∩N(v))
    P := P \\{v}
    X := X ∪ {v}

pivot: -
pivot 版の動作を確認できます
BronKerbosch2(R, P, X):
  if P empty and X empty:
    report R
  choose pivot u in P∪X
  for v in P \\ N(u):
    BronKerbosch2(R∪{v}, P∩N(v), X∩N(v))
    P := P \\{v}
    X := X ∪ {v}
ポイント: 基本形は P のすべての頂点を試しますが、pivot 版は P \ N(u) だけを試します。 黄色で示す「スキップ」頂点は、pivot u の隣接頂点なので、今回は探索しなくても大丈夫です。
